Project Overview

This project implements an intelligent predictive maintenance system for electric boiler equipment, utilizing machine learning algorithms to forecast potential failures and optimize maintenance schedules.

The system processes real-time sensor data from electric boilers including conductivity, pressure, and water level measurements to provide actionable insights for maintenance teams.

Key Features

  • • Real-time sensor data monitoring (conductivity, pressure, water level)
  • • Multi-model ML approach for comprehensive analysis
  • • Interactive dashboard with real-time alerts
  • • Automated maintenance scheduling based on predictions
  • • Historical data analysis and trend visualization

Machine Learning Models

XGBoost Regression

Used for forecasting sensor values and predicting equipment performance degradation over time. Provides accurate predictions of conductivity, pressure, and water level trends.

Isolation Forest

Detects anomalies in sensor readings that may indicate potential equipment issues. Identifies unusual patterns in conductivity, pressure, and water level data.

Random Forest Classifier

Predicts equipment failure probability based on historical data patterns. Classifies equipment states and provides failure probability scores for maintenance planning.

Data Sources

Conductivity Sensor

Measures water conductivity to monitor water quality and detect contamination or scaling issues.

Pressure Sensor

Monitors system pressure to detect leaks, blockages, or pump failures in real-time.

Water Level Sensor

Tracks water levels to prevent overflow or dry running conditions that could damage equipment.

Dashboard Features

  • • Real-time sensor data visualization with historical trends
  • • ML model performance metrics and accuracy scores
  • • Anomaly detection alerts with severity levels
  • • Failure probability indicators and maintenance recommendations
  • • Equipment health score and remaining useful life estimation
